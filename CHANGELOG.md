# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-03-19

### Added
- AI-powered essay pre-grading with 17 evaluation modes
- Teacher review workflow: Accept / Modify / Reject per rubric criterion
- Quiz auto-grading (multiple choice, true/false, checkboxes, short answer)
- Manual grading for quiz paragraph questions with inline scoring UI
- Grade finalization for quiz submissions
- Gradebook with batch grade release and recall
- Student grade view with rubric breakdown
- Rolling release mode (auto-release on confirm)
- PII stripping before AI processing
- Evidence validation for AI-quoted text
- Prompt injection detection
- GDPR-compliant AI data audit logging
- Grade state machine with enforced transitions
- Confidence meters for AI suggestions
- Evidence highlighting in submission text
- Pre-baked demo data (works without API key)
- Groq (Llama 3.3 70B) and OpenAI (GPT-4o-mini) provider support
- Notion-inspired UI with fluid animations (Framer Motion)
- Responsive sidebar layout
- Demo accounts (teacher + 3 students)
- Docker Compose for local PostgreSQL
